# PROJECT365 - LD

Real-time Crypto Market Intelligence Dashboard

Dashboard trading crypto modern dengan tema cyber-gold, fokus pada Market Intelligence, Scanner, Auto Flow Detector, Route Strength, dan Market Sessions.


✨ Fitur Utama

- Live Market Pulse 
- Binance Top Mover Scanner
- Auto Flow Detector (sedang dikembangkan)
- Market Sessions & Killzones
- Sentiment Gauge (Fear & Greed)
- Smart Watchlist
- Responsive & Mobile Friendly
